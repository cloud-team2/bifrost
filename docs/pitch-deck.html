왜 데이터 파이프라인이 필요했을까?

Netflix
목표이탈 방지 — 실시간 추천·부정탐지
데이터가 수많은 마이크로서비스에 흩어져 — 모으지 않으면 추천이 낡고, 계정 도용을 놓친다
→ 구축변경분(CDC)을 실시간으로 한곳에 모으는 파이프라인 직접 구축
NH농협
목표초개인화 + 실시간 이상거래탐지(FDS)
고객 데이터가 1,110개 법인에 흩어져 — 통합 못 하면 한 고객을 못 보고, 야간 배치로는 초 단위 사기를 놓친다
→ 구축1초 미만 통합 상용 CDC(오라클 GoldenGate) 도입
서비스가 커지고 새로 늘어날수록 — 데이터를 잇는 일은 더욱 절실
